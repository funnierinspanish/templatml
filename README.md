# Templatml

1. Paste an HTML template.
1. Use curly braces `{` `}` to wrap variables you want to replace.
1. Edit the form fields to match the variable names.
   1. If a form's input field does not have an associated variable in the HTML template, its left border color will change.
1. Fill out the form.
1. Visualize the rendered HTML content in the "Live Preview" section.
1. Copy the final HTML to the clipboard or download it as an HTML file.
1. Done.
